## Abstract

Microplastic pollution poses one of the most pervasive and undercharacterized threats to global ecosystems. Conventional laboratory workflows вҖ” manual microscopy and bench-top spectroscopy вҖ” are slow, operator-dependent, and impossible to scale for real-world monitoring programs.

**EcoTrack AI** addresses this gap with a reproducible, end-to-end multi-modal AI pipeline:

| Modality | Method | Output |
|---|---|---|
| рҹ”¬ Microscopy Images | YOLOv8 Object Detection | Particle count & bounding boxes |
| рҹ“Ҳ Raman Spectroscopy | Random Forest Classifier | Polymer type & confidence |
| рҹҢҚ Geo-Coordinates | DBSCAN Spatial Clustering | Pollution hotspot maps |
| рҹ–ҘпёҸ Interactive UI | Streamlit Dashboard | Unified visual analytics |

The result is a **high-throughput, automated, and scientifically reproducible** microplastic monitoring framework deployable in laboratory and field settings.

## Data Preprocessing

### Computer Vision Pipeline

```
Raw Image (any size)
       в”Ӯ
       в–ј
Letterbox Resize в”Җв”Җв”Җв”Җв”Җв”Җв–ә 640 Г— 640 (aspect-ratio preserved)
       в”Ӯ
       в–ј
Mosaic Augmentation в”Җв”Җв”Җв–ә Combines 4 images for richer context
       в”Ӯ
       в–ј
Random Horizontal Flip в–ә 50% probability
       в”Ӯ
       в–ј
Brightness / Contrast в”Җв–ә Stochastic jitter for robustness
       в”Ӯ
       в–ј
YOLO Format Labels в”Җв”Җв”Җв”Җв–ә Normalized (x_center, y_center, w, h)
```

### Raman Spectroscopy Pipeline

```
Binary .jws File
       в”Ӯ
       в–ј
Byte Decoding в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв–ә Raw intensity array extraction
       в”Ӯ
       в–ј
Savitzky-Golay Filter в”Җв”Җв–ә Noise suppression (smoothing)
       в”Ӯ
       в–ј
Baseline Correction в”Җв”Җв”Җв”Җв–ә Fluorescence background removal
       в”Ӯ
       в–ј
Min-Max Normalization в”Җв”Җв–ә Intensity range вҶ’ [0, 1]
       в”Ӯ
       в–ј
Feature Extraction в”Җв”Җв”Җв”Җв”Җв–ә Peak positions, widths, ratios
```

## Usage

### рҹ–ҘпёҸ Launch the Streamlit Dashboard

```bash
streamlit run app.py
```

Then open `http://localhost:8501` in your browser.

### рҹ”Қ Run Inference (YOLOv8)

```python
from ultralytics import YOLO

model = YOLO("models/best.pt")
results = model("data/raw/images/sample.jpg")
results[0].show()
```

### рҹ“Ҳ Classify Raman Spectrum

```python
import pickle
import numpy as np

with open("models/raman_classifier.pkl", "rb") as f:
    clf = pickle.load(f)

# features: array of spectral features (see src/signal_proc.py)
features = np.load("data/processed/example_features.npy")
polymer = clf.predict([features])
print(f"Detected Polymer: {polymer[0]}")
```

### рҹ—әпёҸ Run Spatial Clustering

```python
from src.clustering import run_dbscan_hotspots

hotspots = run_dbscan_hotspots(
    coords_csv="data/processed/master_dataset.csv",
    eps=0.01,
    min_samples=5
)
hotspots.plot()
```
